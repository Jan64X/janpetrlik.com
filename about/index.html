About

Third-year high school student in Czech Republic. I run infrastructure, automate things with Ansible, and sometimes take apart IoT devices to see what they're actually doing.

What I'm Doing

Real Infrastructure

Managing production VPS for friends and co-administering the school media team's infrastructure. Real users, real uptime requirements, real backups.

Homelab

A few VMs on a Dell R620 (dual Xeon E5-2670 v2, 256GB RAM). Running TrueNAS, Immich, Navidrome and a Prometheus/Grafana monitoring stack plus other stuff. Everything deployed via a 11k-line Ansible playbook because manual configuration doesn't scale well.

Networking

MikroTik RouterOS for routing, VLANs for segmentation, Hurricane Electric for IPv6. Managing a /48 prefix for IPv6 in GNS3. Hurricane Electric IPv6 Sage certified, which mostly proves I can read documentation ;).

Hardware Hacking

Reverse engineered Xiaomi cameras via UART and modified firmware to add SSH and RTSP. Now they work locally without phoning home. Also reverse engineered Solax solar monitor protocol and wrote a Rust parser for it. If it has a serial port and questionable cloud dependencies, I'm probably interested.

Development

Contributing to Pine64 PineNote development. Python and Bash for automation, learning C for systems programming, dabbling in Rust for random projects and C++ for embedded work on PineTime.
